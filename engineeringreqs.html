Engineering Requirements

Necessary items

  • Autonomous: Once set, must complete task unaided
  • Inexpensive: A relative term, needs to be attainable on a student budget
  • Quiet: Goal of 60dB, about conversational noise
  • Robust: Works for any board layout consistently
  • Fast: Subjective, can it sort in a reasonable amount of time?
  • Efficient: Each placement should use the shortest path possible
  • Safe: Avoids pinch points, expsed wiring
  • User Friendly: Operates with a set and forget ease

Additional requirements involving gameplay

  • Only a single game piece (washer) may be moved at a time
  • A QR code will determine the final location of the game pieces
  • Must contain a single closed loop control system
  • Demonstrate smooth motion
  • LED indication of process status
  • Return to predefined position upon completion
